## 录制操作清单

### 录制前准备
- [ ] 终端字体调大 (18pt+)，深色背景
- [ ] 关闭桌面通知、弹窗
- [ ] 清理终端历史 (`clear`)
- [ ] 确认 `pip install networkx numpy` 已装好
- [ ] 准备一个干净的 git clone（删除旧目录重来）
- [ ] 录屏软件: OBS Studio (免费) 或系统自带

### 录制软件推荐
| 工具 | 平台 | 特点 |
|:---|:---|:---|
| OBS Studio | Win/Mac/Linux | 免费、专业、可设置场景 |
| macOS 自带 | Mac | `Cmd+Shift+5` 直接录 |
| Xbox Game Bar | Win10/11 | `Win+G` 直接录 |

### 配音方案
| 方案 | 工具 | 效果 |
|:---|:---|:---|
| A: 真人录音 | 麦克风 + Audacity | 最自然 |
| B: TTS 合成 | 剪映"文本朗读" | 快、免费、可选音色 |
| C: Edge TTS | `edge-tts` Python 库 | 微软晓晓/云希音色，自然度高 |
| D: 你已有的 IndexTTS2 | RunningHub 工作流 | 你之前配置的音色 |

### 剪辑要点
- 片头 5 秒黑底标题
- 核心镜头(3) 不要加速，让观众看清每一步输出
- 非核心操作 (cd/pip install) 可 2x 加速
- 关键输出行加字幕/高亮框
- 片尾加 GitHub 链接 + Star 引导
